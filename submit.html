Submit Your Game

We are currently taking submissions for HTML/JavaScript Games. However, in order to submit your game, there's a few things you'll need to do:

We use a framework so that any game that follows it will be compatible with our server.

  • Make sure your game has some kind of a score system.

  • Make your javascript class extend the following Game class: Game class

  • Use the User class: User class

If you're not sure how to do so, you're also welcome to right click on one of our games, and click "View Source" to see how the other games use these frameworks.
